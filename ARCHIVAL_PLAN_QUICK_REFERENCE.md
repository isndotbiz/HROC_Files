## EMERGENCY CONTACTS

**Litigation Hold:**
If litigation is anticipated, IMMEDIATELY:
1. Stop all document disposal
2. Notify all personnel
3. Preserve all relevant documents
4. Consult legal counsel
5. Issue formal litigation hold notice

**Lost/Stolen Documents:**
1. Notify Secretary immediately
2. Document what was lost
3. Check backups for electronic copies
4. Request duplicates from issuing agencies if applicable
5. File police report if stolen
6. Review security procedures

**Disaster Recovery:**
1. Assess damage to physical documents
2. Restore from backups (electronic)
3. Request duplicates of critical documents
4. Document loss for insurance/IRS
5. Review and update backup procedures

## QUICK DECISION TREE

**"Should I keep this document?"**

1. Is it a permanent record? (Articles, Bylaws, Board Minutes, Tax Returns)
   - YES → Keep forever, archive when space needed
   - NO → Go to #2

2. Is it a policy, contract, or financial record?
   - YES → Keep 7 years, then destroy per disposal log
   - NO → Go to #3

3. Is it marketing, email, or general business?
   - YES → Keep 3 years, then destroy
   - NO → Go to #4

4. When in doubt:
   - Ask Secretary
   - Err on side of keeping it
   - Review retention policy
   - Document your decision

**"Can I destroy this document?"**

1. Has retention period elapsed? → NO = Keep it
2. Is there any litigation pending/anticipated? → YES = Keep it
3. Is it a permanent record? → YES = Keep it
4. Has Secretary authorized disposal? → NO = Keep it
5. Is it properly logged in disposal log? → NO = Don't destroy yet
6. All YES/NO correct? → OK to destroy per approved method
