# Team Photos Directory

## Required Images

Place the following professional photos in this directory:

### Leadership Team Photos
1. **photo-team-director.jpg** - Executive Director (Indigenous female, professional headshot)
2. **photo-team-outreach.jpg** - Outreach Coordinator (Latino male, professional headshot)
3. **photo-team-peer.jpg** - Peer Support Specialist (Black female, professional headshot)

### Photo Specifications
- **Format**: JPG or PNG
- **Dimensions**: Minimum 400x400px (square aspect ratio preferred)
- **Quality**: High resolution, professional lighting
- **Background**: Neutral or slightly blurred
- **Style**: Professional but warm and approachable
- **File size**: Optimized for web (under 200KB per image)

### Image Sources
You can use:
- **Stock photo sites**: Pexels, Unsplash, Pixabay (search for diverse professional headshots)
- **AI-generated images**: Midjourney, DALL-E, Stable Diffusion (with proper licensing)
- **Professional photography**: Actual team member photos (preferred when available)

### Placeholder Behavior
- If images are missing, the page will display emoji placeholders
- Once you add the photos with the correct filenames, they will automatically appear
- Ensure photos represent the diversity and professionalism described in the bios
