## Web Optimization Checklist

### Before Deployment
- [ ] **Compress all images** using TinyPNG or ImageOptim
  - Target: 100-500KB per image
  - Maintain visual quality

- [ ] **Create WebP versions** for modern browsers
  - Use <picture> element with PNG fallback
  - ~30% smaller file sizes

- [ ] **Implement lazy loading**
  - Use loading="lazy" attribute
  - Defer below-fold images

- [ ] **Add responsive sizing**
  - Use srcset for different screen sizes
  - Serve appropriate sizes for mobile/tablet/desktop

- [ ] **Verify alt text**
  - All images have descriptive alt text
  - Alt text is contextual and meaningful

- [ ] **Test accessibility**
  - Screen reader compatibility
  - Color contrast for overlaid text
  - Keyboard navigation

- [ ] **Performance testing**
  - Page load times under 3 seconds
  - Lighthouse score 90+ for performance
  - Mobile optimization verified
