## âš ï¸ Response Guidelines

**When IRS contacts you:**

### 1. Phone Calls

The IRS now **calls applicants** before sending written requests:
- **First call:** Verify contact information
- **Second call:** 3 days before response deadline as reminder

**If IRS calls:**
- [ ] Ask for agent's name and badge number
- [ ] Ask for callback number to verify legitimacy
- [ ] Take detailed notes
- [ ] Confirm deadline in writing
- [ ] Log in this tracker immediately

**Verify legitimacy:**
- IRS will never demand immediate payment
- IRS will never threaten arrest
- IRS will always provide written follow-up

---

### 2. Written Requests

**Response timeline:**
- Normal deadline: **21 days** (shorter now - check your letter)
- Extension: **14 days or less** based on circumstances
- No extensions for minor requests

**How to respond:**

1. **Format Requirements:**
   - Put organization name and EIN (39-3295288) on each page
   - Identify Part and Line number for each response
   - Consolidate all documents into single PDF (Pay.gov requirement)

2. **Submission Method:**
   - Via Pay.gov if supplementing electronic application
   - Via mail with cover letter to EO Correspondence Unit

3. **What to include:**
   - Cover letter referencing request date
   - Point-by-point responses to all questions
   - Supporting documentation
   - Signature of authorized officer

---

### 3. Notices

**Common IRS notices:**
- **CP-575:** EIN assignment (keep permanently)
- **CP-120:** Request for additional information
- **Determination Letter:** Approval or denial of exempt status

**When you receive a notice:**
- [ ] Read carefully and completely
- [ ] Note all deadlines
- [ ] Log in this tracker
- [ ] Determine if professional help needed
- [ ] Respond by deadline (never ignore!)
